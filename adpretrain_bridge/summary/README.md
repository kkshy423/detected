# 实验汇总模板

命名格式

`日期_方法_数据类型_阶段`

推荐示例

`20260518_ahl_plain_val_threshold_S4`

`20260518_yolo26s_cls_val_threshold_S8`

输出最小字段

`stage status reason accuracy precision recall time_ms f1 auc_roc auc_pr`

空格分隔，不要额外符号，不要长句。

推荐汇总口径

`strategy_stage_adaptive`

`production_normal_p95`

`val_best_f1`

`test_best_f1_oracle`
